# Hvítá, Kljáfoss
Gögnin sem notuð voru í þetta verkefni koma úr LamaH-Ice gagnasettinu (útgáfu 1.5), eftir: Hörður Bragi Helgason.
Hægt er að keyra kóðann með python forritunartungumálinu og með því að hlaða inn LamaH-Ice gagnasettinu í viðeigandi forrit.
Allir 8 liðirnir keyrast sjálfstætt og birta sínar eigin myndir.
